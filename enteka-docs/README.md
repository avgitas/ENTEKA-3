# 📚 ΕΝΤΕΚΑ Knowledge Base (Βάση Γνώσης)

Αυτός ο φάκελος φτιάχτηκε ειδικά για να βάζεις **έγγραφα, σημειώσεις, PDF, Excel ή κείμενα** που θέλεις να θυμάται το AI (Antigravity). 

### Τι μπορείς να βάλεις εδώ:
- 📋 **Το Μενού / Τιμοκατάλογος:** Για να ξέρω ακριβώς τι πουλάει το μαγαζί και σε τι τιμές.
- 📖 **Εγχειρίδια (Manuals):** Οδηγίες για τον εξοπλισμό (π.χ. πώς καθαρίζεται η μηχανή του καφέ), ώστε να μπορώ να βοηθάω τους υπαλλήλους.
- 🧑‍🍳 **Συνταγές / Δοσολογίες:** Πόσα γραμμάρια καφέ μπαίνουν στον Freddo Espresso, τι μπαίνει στο τοστ, κ.λπ.
- 📜 **Κανόνες του Μαγαζιού:** Π.χ. "Όταν κλείνουμε, πάντα μετράμε πρώτα το ταμείο και μετά καθαρίζουμε".
- 📦 **Λίστες Προμηθευτών:** Επαφές, τηλέφωνα, ημέρες παράδοσης.

### Πώς λειτουργεί:
Μόλις ρίξεις ένα αρχείο εδώ μέσα, γράψε μου στο chat: 
*"Έβαλα τον τιμοκατάλογο στο enteka-docs, ρίξε του μια ματιά."*
Αμέσως θα τον διαβάσω, θα τον "απομνημονεύσω" και θα τον χρησιμοποιώ από εδώ και πέρα για ό,τι με ρωτάς!
